A minimal &amp; stateless assert package for writing tests.

## API

```go
assert.Nil(t, nil)
assert.True(t, true)
assert.Equal(t, "Hello", "Hello")
assert.DeepEqual(t, "Hello", "Hello")
assert.Contains(t, "Hello", "ello")
```

## Features

* 0 dependencies
* About 250 lines of code
* Minimal interface
